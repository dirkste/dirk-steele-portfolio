# Personal Developer Website

Static multi-page portfolio site built with HTML, CSS, and vanilla JavaScript.

## Pages

- Home
- About Me
- Work Experience
- Skills & Tech Stack
- Project Showcase
- Publications
- Contact

## Deploy

### GitHub Pages

1. Push the contents of this folder to a repository.
2. In repository settings, enable GitHub Pages for the default branch and root folder.
3. Replace all `https://github.com/YOUR_USERNAME` placeholders with your real profile URL.

### Vercel

1. Import the repository into Vercel.
2. Use the default static site settings.
3. Deploy without a build command.

## Customize

- Replace placeholder text in the HTML files.
- Update the GitHub links in navigation, project cards, and footer.
- If you want the contact form to submit, connect it to a backend or a form service.
